# Synapse
> An experimental programming language written in Python. Synapse purpose is to be simple to master, and, at the same time efficient to code any sort of projects.

 __EN:__
Synapse is a programing language developed in python, it has 3 versions:

* :brain: Interpreted version (almost complete)

* :gear: Compiled version (still in development)

* :page_facing_up: DataBase version (still in development)

## Interpeted Version: 
You can try the main Synapse interpreter by editing the example file:

The synapse main version can be tried by editing the example file: "Synapse_interpreter\exemplo.syn"; You can take a look at the docs ("Synapse_interpreter\extras\documentation_syn.txt");

```
Synapse_interpreter\exemplo.syn
```
```
Synapse_interpreter\extras\documentation_syn.txt
```

## Compiled Version: 
The compiler will translate `.syn` source files into **custom bytecode**, which will be executed by a virtual machine. This part of the project is still under active development.

## DataBase Version: 
This version is similar to `.sql`, where you can save your data in an `.db` file (in synapse case it'll be a `.syn.db` file). The idea of the database version is to be easier than the regular SQL and the database file lighter.

## Synapse front-end
A web-based playground is planned, allowing users to write and execute Synapse code directly in the browser.

# Synapse
> Uma linguagem de programação experimental escrita em Python. O objetivo do Synapse é ser simples de aprender e, ao mesmo tempo, eficiente para codificar qualquer tipo de projeto.

__PT:__
Synapse é uma linguagem de programação desenvolvida em Python e possui 3 versões:

* :brain: Versão interpretada (quase completa)

* :gear: Versão compilada (ainda em desenvolvimento)

* :page_facing_up: Versão de banco de dados (ainda em desenvolvimento)

## Versão Interpretada:
Você pode experimentar o interpretador principal do Synapse editando o arquivo de exemplo:

A versão principal do Synapse pode ser experimentada editando o arquivo de exemplo: "Synapse_interpreter\exemplo.syn"; Você pode consultar a documentação ("Synapse_interpreter\extras\documentation_syn.txt");

``` 
Synapse_interpreter\exemplo.syn
```
```
Synapse_interpreter\extras\documentation_syn.txt
```

## Versão Compilada:
O compilador traduzirá os arquivos de origem `.syn` em **bytecode personalizado**, que será executado por uma máquina virtual. Esta parte do projeto ainda está em desenvolvimento ativo.

## Versão com Banco de Dados:
Esta versão é semelhante ao `.sql`, onde você pode salvar seus dados em um arquivo `.db` (no caso do Synapse, será um arquivo `.syn.db`). A ideia da versão com banco de dados é ser mais fácil de usar do que o SQL tradicional e o arquivo de banco de dados ser mais leve.

## Interface do Synapse
Um ambiente de testes baseado na web está planejado, permitindo que os usuários escrevam e executem código Synapse diretamente no navegador.
